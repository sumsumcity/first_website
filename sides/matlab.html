Kommando-Fenster kann auch wie ein Terminal gebraucht werden. Auchtung Matlab ist vorallem für Naturwissenschaftler. Darum beginnt man bei 1 zu zählen.

Code Erklärung
help help Zeigt auf was man machen kann
a = 2.5 Variabeln speichern
a = 2.5; Ohne Rückmeldung
a= 1.7, b=2.45 Kommas trennen die Befehle
clear a Löscht die Variable a
clear all Löscht alle Variabeln
clc Säubert das Kommando-Fenster
whos a Zeigt was und wieviel stoarge die Variable braucht

Vectorial computations

Matlab stores every number in an array or matrix

1xn or nx1 are both vectors

Code Erklärung
a=[1,2,3,4] a=[1 2 3 4] row vecor (1x4)
a=[1;2;3;4] column vector (4x1)
a = beginning number:step size:ending number creates vetor with these values
b=zeros(1,10) b=ones(1,10) b=4*ones(1,10) fills vector with number
b = linspace(1,10,3) [1.0 5.5 10.0]
a(3) a(end) a(2:7) takes third or end element in vector a
size(a) length(a) size shows rows and columns
