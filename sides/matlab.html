Vectorial computations

Matlab stores every number in an array or matrix

1xn or nx1 are both vectors

Code Erklärung
a=[1,2,3,4] a=[1 2 3 4] row vecor (1x4)
a=[1;2;3;4] column vector (4x1)
a = beginning number:step size:ending number creates vetor with these values
b=zeros(1,10) b=ones(1,10) b=4*ones(1,10) fills vector with number
b = linspace(1,10,3) [1.0 5.5 10.0]
a(3) a(end) a(2:7) takes third or end element in vector a
size(a) length(a) (max(size(a)) | size(a, 1))) size shows rows and columns
+ - * / +/- If they have the same length. * Matrix multiplication. / is right division so: A/B=C --> C*B=A
b = a' Transpose
a+b (not same Orientation) Broadcasting
c = a.*b | c = a*b’ | c = b’*a Element-wise multiplication | Scalar product | Broadcast multiplication
c = a./b Element-wise usual division

Matrices

Matlab stores every number in an array or matrix

1xn or nx1 are both vectors

reshape(a,2,6) macht bestehende matrix zu 2x6 Matrix
